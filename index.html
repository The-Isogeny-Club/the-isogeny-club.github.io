Upcoming talks

  1. git commit -m “isogenies”
    Bruno Sterner, University of Surrey
    September 27th, 17:00 (CEST).
    abstract

    Supersingular isogeny graphs possess many properties that make it an interesting object to study mathematically as well as attempt to apply for cryptographic purposes. In this talk, I will present one of these properties and showcase how it can be applied to construct a commitment scheme. This commitment scheme has strong security properties and doesn’t require random oracles.
